# AI Research & Learning Companion

### An Agentic Learning Intelligence System

> Not a chatbot  
>  A system that learns how _you_ learn

---

## Project Overview

The **AI Research & Learning Companion** is an **Agentic AI system** designed to actively support learning and research by adapting to the user over time.

Unlike traditional AI tools that only respond to prompts, this system:

- Understands what the user is learning
- Diagnoses conceptual gaps
- Adapts explanations dynamically
- Actively questions the learner
- Builds long-term learning memory

This is a **learning intelligence**, not a static assistant.

---

## Problem Summary (Why This Matters)

Modern learners face three major issues:

1. **Information Overload**  
   Too many papers, blogs, videos — no structure.

2. **Passive Learning**  
   Existing AI explains but does not _evaluate understanding_.

3. **No Learning Memory**  
   Current tools forget what the user already knows or misunderstands.

### Critical Gap

There is no AI system that **tracks conceptual understanding and evolves with the learner over time**.

---

## Agentic System Design

This project uses a **multi-agent architecture**, where each agent has a clear responsibility.

### Agent List (5 Core Agents)

1. **Learning Diagnosis Agent**  
   Identifies user knowledge level and gaps.

2. **Concept Explanation Agent**  
   Generates explanations adapted to the learner’s depth.

3. **Socratic Questioning Agent**  
   Asks follow-up questions to test understanding.

4. **Memory & Progress Agent**  
   Stores learning history, weaknesses, and mastery scores.

5. **Adaptation Strategy Agent**  
   Adjusts teaching strategy based on past performance.

---

## System Flow (High-Level)

1. User asks a question
2. Diagnosis Agent evaluates understanding
3. Explanation Agent responds
4. Socratic Agent asks probing questions
5. Memory Agent updates learning profile
6. Adaptation Agent modifies future behavior

Each interaction improves the system’s understanding of the learner.

## Why This Is **Agentic AI** (Not a Chatbot)

This system qualifies as **true Agentic AI** because:

- Multiple specialized agents
- Autonomous agent decision-making
- Persistent learning memory
- Behavior changes over time
- Agents interact and evaluate each other
